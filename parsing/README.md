# Parsing smtp message
To test the programm, enter
```bash
python -m venv venv/
. ./venv/Scripts/activate
pip install -r requirements.txt
python parsing.py test/smtp_test.txt
```
with exemple_smtp_attachment.txt the smtp session

```bash
python -m venv venv/
```
